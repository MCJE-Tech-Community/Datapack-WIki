# MC-Command-Ideas
メモ用に比較的小規模なコマンドネタをまとめたものです。  
一つのデータパックとして配布するには小さすぎる物やモジュール化が面倒な物などを、個人的なメモ代わりとして置いておくためのリポジトリです。コピペや改造しての利用は構いません。メモ程度なので説明不足な点や煩雑な書き方になっている部分もあるかもしれませんが悪しからず。
